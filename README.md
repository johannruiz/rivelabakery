# Rivela Web

Sitio estatico para catalogo, carrito y pedido por WhatsApp.

## Probar en localhost

En Windows, abre:

```text
start-localhost.bat
```

Luego entra en el navegador a:

```text
http://localhost:8000/
```

Para detener el servidor, vuelve a la ventana de la terminal y presiona `Ctrl + C`.

## Productos

La base de productos esta en:

```text
data/products.json
```

Edita ese archivo para agregar, quitar o cambiar productos.
